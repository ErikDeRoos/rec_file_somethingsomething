# 9_int_field_type

## Purpose
This example demonstrates a valid `int` field type declaration and compatible values.

## What it demonstrates
- `%type: Priority int`
- decimal integer values
- negative integer values
- hexadecimal integer values using the `0x` prefix
- octal integer values using a leading `0`

## Files
- `tasks.rec` - valid example showing records with an `int`-typed field
